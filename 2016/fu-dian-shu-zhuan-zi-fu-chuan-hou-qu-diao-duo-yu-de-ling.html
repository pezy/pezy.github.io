浮点数转字符串后, 去掉多余的零

gist · 2016-04-25 · pezy

当数据用于显示的时候, 经常会遇到这样类似的需求. 今天先用 C++ 给一个简单的实现:

std::string trimToString(double dValue)
{
    std::string strRst = std::to_string(dValue);
    while (strRst.back() == '0') strRst.pop_back();
    if (strRst.back() == '.') strRst.pop_back();
    return strRst;
}
